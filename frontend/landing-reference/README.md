# Cursor Local — Landing Page

Landing page profissional para a plataforma Cursor Local.

## Executar

```bash
cd landing
npm install
npm run dev
```

Acesse: **http://localhost:5174**

## Build

```bash
npm run build
npm run preview
```

## Stack

- React + TypeScript + Vite
- TailwindCSS v4
- Framer Motion
- Lucide Icons

## Seções

- Hero com mockup do IDE
- Recursos (6 cards)
- Como Funciona (4 etapas)
- Comparativo de benefícios
- Demonstração animada
- Depoimentos
- FAQ (8 perguntas)
- CTA final
